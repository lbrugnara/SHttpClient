SHttpClient
===========

Simple HTTP Client is a tiny object-oriented PHP library to transferring data using [PHP cURL](http://php.net/manual/en/book.curl.php)

###### Version 0.1.0
 - Support for GET, POST, PUT, DELETE and HEAD HTTP methods
 - Multipart messages
 - SSL support

###### - Requirements
 - PHP 5 >= 5.4
 - cURL extension
